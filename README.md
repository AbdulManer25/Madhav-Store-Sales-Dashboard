# 📊 Madhav Store Sales Dashboard

An interactive Power BI dashboard project for analyzing sales, profit, customer behavior, and business performance.

---

## 🚀 Features

- Sales & Profit Analysis
- Customer Insights
- Product Performance Tracking
- Region-wise Analysis
- Interactive Filters & Visuals

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Excel / CSV
